## express-blog

npm install 

node app.js

### Folder Structure

The default file and folder structure is as follows:

* README.md
* app.js
* package.json
* assets
    * css
        * Any CSS Files
    * img
        * Any Image Files
    * js
        * Any Client-side JS files
    * less
        * skeleton.less
* views
    * index.html
    * error.html
    * partials
        * head.html
        
The rendered file structure would be similar to:

* /
* /css/skeleton.css
* /img/someImageFile.png
* /js/someClientJS.js
